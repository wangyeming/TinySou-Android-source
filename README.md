TinySou-Android-source
======================

TinySou-Anddroid TinySouHelp.jar 源码
-------------------------------------

包含三个class文件，分别为HttpHelp, TinySouHelp, TinySouClient三个自定义类

HttpHelp
--------
Java http帮助类，执行http请求，监听http状态等。

TinySouHelp
-----------
TinySou帮助类，解析微搜索返回Json数据。

TinySouClient
-------------
封装微搜索请求过程，返回微搜索结果
